struct

struct(结构体)的定义

struct(结构体)是一种聚合类型，里面可以包含任何类型的值，这些值就是我们定义的结构的成员，也称为字段

关键字和声明

type+struct

示例

使用

  • 结构体里面值的类型可以为结构体
  • 给变量赋值结构体时内容可以为空，这样所有字段为空

由于golang没有python和c++那样的class用来定义类，所以在golang中，可以使用结构体完成面对对象

interface

interface(接口)的定义

interface(接口)是与调用方的一种约定，它是一个高度抽象的类型，不用与具体的实现细节绑定在一起。接口要做的是定义好约定,告诉调用方自己可以做什么，但调用方不用直到它内部的实现，这与我们见到的具体类型如int, map, slice等不一样。

关键字和声明

关键字type interfaceName interface

个人理解

接口其实就是一个中转站，这里放着可以调用的方法，然后结构体，类等要实现对应的方法，否则无法编译通过。

只要接口的约定不变，接口的实现者可以更好地升级重构。
